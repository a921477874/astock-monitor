# 五维度市场分析框架 — 宝宝量化系统

> 2026-07-18 建立，基于2.65万亿量价误判教训

## 核心理念

**市场走势不是单变量函数。** 任何单一维度（只看涨跌、只看成交额、只看技术指标）都会导致误判。
必须5个维度交叉验证，结论一致才能下判断，有分歧时结论要保守。

---

## 一、量价维度（最基础，但也最容易犯错）

### 数据源注意
| 字段 | 接口 | 单位 | 常见错误 |
|:----|:-----|:----:|:--------|
| 上证成交额 | `qt.gtimg.cn/q=sh000001` 字段[37] | **万元** ÷10000得亿 | 误当元处理（踩坑过） |
| 深证成交额 | `qt.gtimg.cn/q=sz399001` 字段[37] | **万元** ÷10000得亿 | 同上 |
| **全市场成交额** | = 上证 + 深证 | 亿元 | 只看了上证单一市场 |
| 涨跌家数 | 新浪全市场分页扫描 | 沪A+深A | 各API采样不全需分页 |

### 量价分析要点
- **放量位置很关键**：低位放量是换手见底信号；高位放量是出货信号
- **缩量程度**：2.65万亿恐慌量→缩到1.6~1.8万亿才算企稳
- **量价背离**：价格反弹但量跟不上→逃命波；价格横盘缩量→蓄势

### 量价五情景

| 情景 | 量 | 价 | 含义 |
|:----|:---|:---|:-----|
| 放量下跌 | ↑↑ | ↓↓ | 恐慌踩踏 or 主力吸筹（看位置） |
| 缩量下跌 | ↓↓ | ↓↓ | 阴跌磨底，最难受但最接近底部 |
| 放量上涨 | ↑↑ | ↑↑ | 有资金接力，可关注 |
| 缩量上涨 | ↓↓ | ↑↑ | 反弹无力，容易回落 |
| 放量震荡 | ↑↑ | → | 分歧大，等方向 |

## 二、技术结构维度（均线+形态+关键点位）

### 均线三线判断
```
股价 > MA5 > MA20 > MA60  → 多头排列（强势）
股价 < MA5 < MA20 < MA60  → 空头排列（弱势）
MA5 ≈ MA20 ≈ MA60 粘合    → 横盘整理，即将变盘
乖离率大（股价远离MA5）    → 短期超买/超卖，有回归需求
```

### 关键位置分析
```
60日最高 — 强阻力区
60日均线 — 中期牛熊分界线
20日均线 — 短线强弱分水岭
5日均线  — 超短线支撑/压力
60日最低 — 强支撑区
52周高/低 — 大周期位置判断
```

### 量价+技术交叉验证
例如：7/17的情况
- 价格跌破所有均线（空头信号）
- 但2.65万亿放量换手（见底的必要非充分条件）
- 乖离率-3.4%（超卖后有反弹需求）
- 结论：矛盾，需要其他维度辨别

## 三、产业/政策维度（A股特色，不可忽略）

### 大IPO压制逻辑
```
超级IPO前1~2月 → 同板块被压制
原因：压低估值→洗浮筹→低位承接新股
案例：长鑫存储→半导体/科创板被杀
对策：手动维护 MAJOR_IPOS 列表，监视期自动降分
```

### 国家队行为
- **托而不举**：只保融资功能，不代表看好市场
- 上证50/沪深300抗跌 ≠ 市场见底（只是国家队在托）
- 上证50跌2.6% vs 科创跌7.1% → 结构性行情不是普涨

### 政策信号分类
| 级别 | 信号 | 影响 |
|:----|:----|:----|
| 🟣 强力利好 | 降准降息/印花税调降/汇金增持 | 可扭转趋势 |
| 🔵 利好 | 产业扶持/减税/开放外资 | 短期提振 |
| 🟢 中性 | 正常监管/常规发审 | 影响有限 |
| 🟡 利空 | 严监管/查配资/收紧流动性 | 压制市场 |
| 🔴 强利空 | 加息/加税/限定行业政策 | 趋势性打击 |

## 四、跨市场维度（内外联动）

### 跨市场传导链
```
美股（纳指/标普）→ 恒生科技 → A股科技股
A股盘后（15:00）→ 欧股开盘 → 美股开盘 → 次日A股
```

### 关键时间窗口
| 时间 | 事件 | 对A股影响 |
|:----|:-----|:---------|
| 周六凌晨 | 美股周五收盘 | 周一开盘情绪锚 |
| 周日全天 | 新闻/政策发布 | 直接情绪影响 |
| 周一09:00 | 期货开盘 | 预判方向 |
| 周一09:25 | 集合竞价 | 真实方向信号 |

### 跨市场判断法则
- 美股跌但A50期货不跟 → 独立性行情，不必恐慌
- 美股跌且A50期货跟跌 → 共振下行，谨慎
- 恒生科技比A股先企稳 → 领先指标，A股可能跟随

## 五、情绪维度（最难量化，但最关键）

### 情绪阶段图谱

| 阶段 | 特征 | 操作 |
|:----|:-----|:----|
| **亢奋** ≥8分 | 涨停潮、散户跑步入场、人人赚钱 | 减仓，准备跑 |
| **活跃** ≥6.5分 | 赚钱效应好、主线清晰 | 积极操作 |
| **中性** ≥5分 | 涨跌互现、没有明确方向 | 等回调建仓 |
| **低迷** ≥3.5分 | 缩量阴跌、没人谈股票 | 防守低吸 |
| **冰点** <3.5分 | 恐慌割肉、绝望、骂市场 | **不出票！** |

### 情绪观察指标
- 跌停板数量 → 恐慌的体温计
- 散户论坛/社交平台语气 → 从愤怒到绝望的周期
- 身边人谈股票的热度 → 大妈开户热度是领先指标
- 成交额极端值 → 天量见顶、地量见底

### 周五（7/17）情绪判断
- 创业板-7.15% → 极致恐慌
- 2.65万亿成交量 → 踩踏换手
- 情绪评分预估在2~3分（冰点）
- 物极必反 → 周一有反弹需求（但不代表反转）

每日分析时，对每个维度打分：🟢积极 / 🟡中性 / 🔴消极

| 维度 | 周五(7/17) | 周一预判 | 权重 |
|:----|:----------:|:--------:|:----:|
| 量价 | 🔴 放量恐慌下跌 | 🟡 超跌→缩量是积极信号 | 25% |

**综合判断**：3🔴2🟡，偏空但有超跌反弹预期。等待缩量企稳信号。

## 七、常见的单维度错误（踩过的坑）

| 错误 | 踩坑日期 | 教训 |
|:----|:--------:|:----|
| 只看上证成交额当全市场 | 2026-07-18 | 全市场=上证+深证 |
| 腾讯[37]单位当元 | 2026-07-18 | 单位是"万"÷10000得亿 |
| 放量杀跌=恐慌（单一看） | 2026-07-18 | 低位放量也可能是换手见底 |
| 单日大跌=完蛋 | — | 冰点后常有暴力反弹 |
| 情绪好=一直好 | — | 亢奋是减仓信号不是追涨信号 |

## 八、使用流程

```
开盘前/复盘时
    ↓
拉取5维度数据（量价/技术/产业/跨市场/情绪）
    ↓
每个维度独立打分（🟢🟡🔴）
    ↓
看是否一致
    ↓
一致 → 按信号操作
分歧 → 保守处理（等待或减仓）
    ↓
判断结论 = 多维度交叉验证的结果
       ≠ 某个单一指标的结论
```
