## 核心铁律

```
1. 大盘评分<3.5 → 不出票，全系统冰点提示
2. 每只推荐带买入区间+止损+止盈
3. 产业逻辑>技术信号 (主力思维，不是散户思维)
4. 每日诚实复盘，记录验证结果
5. 不用管对错，数据说话
```
