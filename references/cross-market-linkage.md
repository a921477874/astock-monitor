# 跨市场联动分析框架

## 核心发现

恒生科技2026-07-17真实跌幅：**-4.32%**（之前误报为-0.06%）
→ 港股和A股周五是**共振下跌**的，不是A股独立暴跌。
→ 这意味着：**这不是纯A股内部的结构性调整，而是有跨市场因素在推动。**

## 共振 vs 独立

| 下跌类型 | 特征 | 后续 |
|:--------|:-----|:-----|
| **A股独立跌**（港股/美股不跌） | 内因驱动，如IPO冲击、政策利空 | 消化快，可能V反 |
| **共振跌**（港股也跌） | 外因+内因叠加 | 需要更长时间修复 |
| **港股先跌A股后跌** | 资金外流/汇率贬值 | A股有补跌空间 |

## 联动机制分类

### 机制1：资金面联动（最直接）
```
美股/港股大跌 → 外资风险偏好下降 
  → 北向资金流出A股
  → A股核心资产（白马/权重）承压
  → 上证50/沪深300下跌 → 拖累全市场
```
**特征**：上证50和恒生指数同步，外资重仓股领跌

### 机制2：情绪面联动（最广泛）
```
美股科技大跌 → 全球科技股情绪恐慌
  → 恒生科技下跌 → A股科技股跟跌
  → 科创50/创业板领跌
```
**特征**：科创50和恒生科技同步，科技板块集体下跌

### 机制3：汇率联动（最持久）
```
人民币贬值 → 外资流出A股+港股
  → 两地同步承压
  → 需要央行干预或经济数据扭转
```
**特征**：持续阴跌，跌速不快但时间长

## 自动化的规则

在 `global_mood.py` 中增强判断：

```python
# 联动强度评分（-5 ~ +5）
# 正=联动强(跟跌)，负=独立(不跟)
linkage_score = 0

# 1. 恒生科技 vs 科创50
hstech_chg = -4.32  # 恒生科技当日涨跌
kc_chg = -7.12      # 科创50当日涨跌
if abs(hstech_chg - kc_chg) < 2:
    linkage_score += 2  # 高度同步→情绪联动强
elif abs(hstech_chg - kc_chg) < 5:
    linkage_score += 1  # 部分同步
else:
    linkage_score -= 1  # 背离→独立行情

# 2. 恒生指数 vs 上证50
if abs(hsi_chg - sh50_chg) < 1:
    linkage_score += 2  # 资金面联动
elif abs(hsi_chg - sh50_chg) < 3:
    linkage_score += 1
else:
    linkage_score -= 1

# 3. 判断类型
if linkage_score >= 3:
    type = "共振下跌"  # 跨市场风险
elif linkage_score >= 1:
    type = "部分联动"  # 有独立因素
else:
    type = "独立行情"  # A股在走自己的逻辑
```
