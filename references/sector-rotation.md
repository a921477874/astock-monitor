# 板块轮动节奏框架

> 2026-07-18 建立

## 核心规律

**资金永远不会同时买所有板块。** 理解资金的流向前后顺序，就能提前布局下一个要涨的板块。

## A股板块轮动时序（从冰点到亢奋）

```
冰点期（评分<3.5）
  ├ 高股息（银行/煤炭/公用事业）→ 最先企稳，资金护盘首选
  ├ 黄金/贵金属 → 避险资金
  └ 科技/成长 → 继续杀估值，不要碰
      ↓
低迷期（评分3.5~5）
  ├ 消费/医药 → 超跌反弹（防御属性能看到业绩）
  ├ 金融（银行/保险/券商）→ 低估值+高股息，机构回补
  └ 科技/成长 → 跌速放缓，但还没到底
      ↓
中性期（评分5~6.5）
  ├ 可选消费（汽车/家电）→ 政策驱动+业绩复苏
  ├ 工业/原材料 → 经济复苏预期
  └ 科技/成长 → 开始有资金试探性建仓
      ↓
活跃期（评分6.5~8）
  ├ 科技（半导体/AI/新能源）→ 主升浪启动🚀
  ├ 券商 → 牛市旗手，市场情绪从谨慎变乐观
  └ 小盘题材 → 游资活跃，赚钱效应扩散
      ↓
亢奋期（评分≥8）
  ├ 所有板块轮动完毕 → 补涨/滞涨股最后拉升
  ├ 新股/次新 → 资金没地方去了
  └ 垃圾股/ST → 狂热期的最后一棒
```

## 关键判断：现在处于哪个位置

**当前（2026-07-17）→ 冰点期**
```
冰点期应该看到：
  上证50（高股息/银行）> 沪深300（蓝筹）> 中证500（中小盘）> 科创50（成长）
  
  7/17实际表现：
  上证50: -2.63%
  沪深300: -3.60%
  中证500: -5.55%
  科创50: -7.12%
  
  ✅ 完全符合冰点期特征：蓝筹最抗跌，成长杀最狠
```

**结论：**
1. 科技/成长（半导体、AI、新能源）短期不是底，不要抄
2. 银行/高股息相对抗跌，但也不是建仓时机（冰点期不出票）
3. **等看到消费/医药先企稳反弹时，才是情绪从冰点→低迷的信号**
4. **等科技开始缩量止跌时，才是可以准备进场的时机**

## 追踪信号

| 信号 | 含义 | 操作 |
|:----|:----|:----|
| 银行股止跌反弹 | 国家队护盘，冰点接近尾声 | 可以关注，但不出手 |
| 消费/医药连续3天不创新低 | 情绪从冰点→低迷 | 小仓试探（2成） |
| 科技板块缩量到地量 | 杀跌动能衰竭 | 准备建仓 |
| 科技板块放量首阳 | 资金回流科技 | 加仓信号🚀 |

## 将轮动框架系统化

在 `sector_flow.py` 或 `update_dashboard.py` 中增加：

```python
轮动阶段判断（根据各指数相对强弱）:
  1. 如果 上证50 > 沪深300 > 科创50 → 冰点/防御阶段
  2. 如果 上证50 ≈ 沪深300 > 科创50 → 低迷/止跌阶段
  3. 如果 科创50 开始 > 沪深300 → 活跃/成长领涨阶段
  4. 如果 小盘/题材 领涨 → 亢奋/尾声阶段

轮动位置输出到六维分析的"板块协同"维度：
- 冰点防御 → 板块协同低分（无主线）
- 成长领涨 → 板块协同高分（有主线）
```
