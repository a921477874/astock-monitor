## 一、宏观周期

### 四阶段

| 阶段 | 特征 | 利率 | PMI | 通胀 | 最适合的资产 |
|:----|:-----|:---:|:---:|:----|:------------|
| **衰退** | GDP增速放缓，企业盈利下滑 | 低位/降息 | <50 | 低 | 债券 > 现金 > 股票 |
| **复苏** | 经济触底回升，政策效果显现 | 低位/持平 | 回升>50 | 温和 | **股票 > 商品 > 债券** |
| **过热** | 经济过热，通胀上升 | 高位/加息 | >52 | 高 | 商品 > 现金 > 股票 |
| **滞胀** | 经济停滞+通胀高企 | 高利率 | <50 | 高 | 现金 > 商品 > 债券 |

### 当前判断（2026-07-18）
- 全球经济放缓，但中国处于**宽松周期**（降准降息窗口）
- PMI在50附近徘徊，尚未明确站上复苏线
- 通胀低位（CPI<1%），为政策留足空间
- → **定位：衰退→复苏的过渡期**（最有利于股市筑底的阶段）

## 二、市场周期

### 六阶段（结合情绪评分量化）

```
评分  阶段        操作          投资者行为
───────────────────────────────────────────
8~10  亢奋        减仓✅        散户跑步入场
6~8   活跃        加仓✅        有主线，赚钱效应
5~6   中性        持仓等        方向不明，换手率低
3~5   低迷        低吸✅        无人问津，割肉离场
1~3   冰点        不出票❌      绝望、恐慌踩踏
0~1   崩溃        清仓/等待      极端行情（千股跌停等）
```

### 当前定位
- 情绪评分：**2.4分（冰点）**
- 阶段：冰点（接近崩溃边缘但还没到）
- 仓位：0~1成（空仓）

### 历史位置对照
| 阶段 | 上证点位参考 | 特征 |
|:----|:-----------|:-----|
| 2024年2月冰点 | 2635 | 雪球爆仓，千股跌停 |
| 2025年4月冰点 | 3050 | 关税冲击，恐慌踩踏 |
| **2026年7月冰点** | **3764** | **长鑫IPO杀估值** |
| → 每次冰点后3~6个月，上证平均反弹+15% | | |

## 三、板块周期

### 轮动顺序（从冰点到亢奋）

```
防御 → 金融 → 消费 → 周期 → 成长 → 题材

冰点期 → 资金去高股息/银行/公用（保命）
低迷期 → 资金去消费/医药/金融（求稳）
中性期 → 资金去周期/制造（博弈经济复苏）
活跃期 → 资金去科技/成长（追弹性）
亢奋期 → 资金去小盘/题材（最后的狂欢）
```

### 当前定位
- **防御阶段**（银行/公用/高股息最抗跌）
- 还没到消费/医药的反弹阶段
- 科技/成长还要继续杀

## 四、个股周期（主力操盘四阶段）

```
吸筹 → 洗盘 → 拉升 → 出货

 吸筹阶段：缩量横盘，均线粘合，主力悄悄买
 洗盘阶段：故意砸破支撑，放量吓散户交出筹码
 拉升阶段：温和放量，连续小阳线，量价齐升🚀
 出货阶段：放量冲高回落，主力边拉边卖
```

### 当前：个股普遍在洗盘阶段
- 周五的2.65万亿放量暴跌，不排除是主力利用IPO利空**
- 恐慌换手后，底部会更扎实

## 四个周期叠加 — 当前总定位

| 周期 | 阶段 | 评分 | 操作 |
|:----|:----|:---:|:----|
| 🏛️ 宏观 | 衰退→复苏过渡 | 🟡 | 政策宽松期，有利于筑底 |
| 📈 市场 | **冰点（2.4分）** | 🔴 | **空仓，不出票** |
| 🏭 板块 | 防御阶段 | 🔴 | 只看高股息，不碰科技 |
| 📉 个股 | 洗盘阶段 | 🔴 | 等待洗盘结束信号 |

**总判断：四个周期中三个处于偏弱状态（市场/板块/个股），宏观处于中性偏暖。** 
→ 还需要时间，不急于抄底。

**何时转向？**
```
宏观 → 持续宽松（已经在做）
市场 → 情绪评分回升到4分以上（还差）
板块 → 消费/医药开始止跌反弹（还没看到）
个股 → 缩量企稳+均线粘合（需要等）
```

等到这四个条件中有**三个**转向，就是可以进场的信号了。
