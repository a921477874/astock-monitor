# 历史相似行情匹配 — 参考框架

> 2026-07-18 建立，手动维护

## 匹配逻辑

每次大盘出现重大波动（单日跌>2%或连续跌>5%），自动匹配历史上最相似的行情。

### 匹配维度

| 维度 | 权重 | 说明 |
|:----|:---:|:-----|
| 单日跌幅 | 25% | 当日上证涨跌幅 |
| 累计跌幅 | 20% | 近5日/10日累计跌幅 |
| 成交额变化 | 15% | 放量/缩量比例 |
| 情绪评分 | 15% | mood_cycle评分 |
| 板块结构 | 15% | 科创vs蓝筹分化度 |
| 触发原因 | 10% | 关键词匹配（IPO、关税、政策等） |

### 历史数据库（手动维护，持续扩充）

每条记录包含：
- 日期范围、上证点位、累计跌幅
- 触发原因（一句话）
- 后续走势（1周/1月/3月后涨跌幅）
- 关键特征标签

## 匹配算法

```python
def match_history(current, history_db):
    """
    current: {
        'sh_change': -3.05,       # 上证当日涨跌%
        'kc_change': -7.12,       # 科创50当日涨跌%
        'total_amt': 26550,       # 全市场成交额(亿)
        'mood_score': 2.4,        # 情绪评分
        'trigger_kw': 'IPO',      # 触发原因关键词
    }
    returns: [(相似度评分, 历史事件), ...] 按相似度降序
    """
    scores = []
    for event in history_db:
        score = 0
        # 单日跌幅匹配
        score += (5 - abs(current['sh_change'] - event['sh_change'])) * 5
        # 科创跌幅匹配（科创板IPO事件特化）
        if 'kc_change' in event:
            score += (5 - abs(current['kc_change'] - event['kc_change'])) * 4
        # 成交额匹配
        score += (5 - abs(current['total_amt']/10000 - event['amt']/10000)) * 3
        # 触发原因关键词匹配
        if current['trigger_kw'] in event.get('tags', ''):
            score += 3
        scores.append((round(score, 1), event))
    return sorted(scores, key=lambda x: -x[0])
```
